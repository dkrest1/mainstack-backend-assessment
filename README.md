- Mongoose ORM

## Getting Started

1. Clone the The Mainstack Backend Assessment Repo

```bash
$ git clone repo url
$ cd mainstack-backend-assessment
```

2. Create a .env file and copy files from .env.example to .env

```bash
$ cp .env.example .env
```

3. install necessary packages

```bash
$ npm install
```

4. Start or Spin up the server

```bash
$ npm run dev
```
